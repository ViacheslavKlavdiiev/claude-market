# Changelog

- Initial release: plugin scaffolded, with 4 Flutter task skills vendored
  verbatim from the official BSD-3 `github.com/flutter/agent-plugins`
  (Copyright 2026 The Flutter Authors) — `flutter-build-responsive-layout`,
  `flutter-implement-json-serialization`, `flutter-setup-localization`,
  `flutter-use-http-package`.
- The five house-authored conceptual skills (`flutter-best-practices`,
  `flutter-architecture`, `flutter-material3`, `flutter-routing`,
  `flutter-testing`) ship in this release.
